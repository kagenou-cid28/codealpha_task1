# Voxora

### Every Voice Understood.

Voxora is a modern language translation workspace built to make multilingual communication simple, fast, and accessible.

Unlike traditional translator applications, Voxora focuses on creating a clean workspace experience inspired by modern productivity tools and developer platforms. Users can translate text, upload documents, listen to translated content, and manage their translation history from a single interface.

Developed as part of the **CodeAlpha Artificial Intelligence Internship – Task 1 (Language Translation Tool)**.

## Project Overview

Language barriers shouldn't slow communication.

Voxora provides a modern translation environment where users can:

- Translate text between multiple languages
- Upload TXT and PDF documents for translation
- Listen to translated content using Text-to-Speech
- Save translation history automatically
- Download translated results
- Switch between languages instantly

The goal was to create something that feels closer to a professional productivity application rather than a simple translation website.

## Architecture

Voxora is intentionally built as a lightweight frontend-first application.

```text
User Input
     ↓
Translation Request
     ↓
Translation API
     ↓
Translated Response
     ↓
History Storage (Local Storage)
     ↓
User Workspace
```

No backend server is required.

All translation history is stored locally in the user's browser.
